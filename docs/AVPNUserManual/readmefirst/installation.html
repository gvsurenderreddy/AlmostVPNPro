Installation Instructions for AlmostVPN

Preference Pane Installation

There are two ways to install a Preference Pane:
  • double-click it, in the finder,
  • copy it into /Library/PreferencePanes or  ~/Library/PreferencePanes
You can use either of these ways to install AlmostVPN Preference Pane.  Unlike 0.9.x version, which must be installed "For this User only" (~/Library/PreferencePanes), version 1.0 (as of GA) could be installed "For All Users" (/Library/PreferencePanes) as well as "For This User".  

NOTE: Though it is possible to install AlmostVPN "For All Users", as of now it is still "single-user" application.  Only one user on particular host can use AlmostVPN.

NOTE: As of "GA", AlmostVPN 1.0 will coexist peacefully with 0.9.x.  You do not have to un-install 0.9.x version before you install 1.0.

NOTE: When installed using "double-click",  it is advisable to re-start System Preferences right after AlmostVPN has been installed.  

 AlmostVPN Menu Bar Installation

AlmostVPN Menu Bar application packaged inside of AlmostVPN Preference Pane.  To install (and start it), go to "Preferences/General" tab and select "Menu Bar App: start at login" option.  Once checked, it will install Menu Bar App as a "login item" to make it start automatically on each login.
